# PO Automation - XII Tel 13

Welcome to the **XII Tel 13 Pre-order Automation Portal**! This modern web application streamlines the pre-order process, from customer browsing to administrative order tracking and management.

## 🚀 Key Features

- **Modern Landing Page**: High-performance, visual product showcase.
- **Pre-order Portal**: User-friendly ordering flow.
- **Admin Dashboard**: Comprehensive order & product management system.
- **Secure Access**: Built-in authentication middleware for dashboard protection.
- **Data Export**: Export orders for easy tracking and fulfillment.

## 🛠️ Quick Start

### 1. Install Dependencies

```bash
npm install
```

### 2. Configure Environment Variables

Create a `.env` file with your Supabase credentials:

```bash
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_key
```

### 3. Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.
