# Project Overview: PO Automation - XII Tel 13

This project is a **Pre-order Automation System** designed for **XII Tel 13**. It provides a seamless interface for customers to browse products and place pre-orders, while offering a comprehensive dashboard for administrators to manage products and track orders.

## Key Goals

- **Automation**: Streamline the pre-order process from customer selection to administrator tracking.
- **Modern UI**: Provide a premium, responsive user experience using modern web technologies.
- **Efficiency**: Enable easy data management and export for order fulfillment.

## Core Features

1. **Public Landing Page**: A visually appealing showcase of available products with real-time stock and pricing.
2. **Pre-order Portal**: User-friendly flow for customers to submit their orders.
3. **Admin Dashboard**:
   - **Performance Overview**: Quick stats on sales and orders.
   - **Order Management**: Detailed list of all pre-orders with search and filter capabilities.
   - **Product Management**: Interface to add, edit, or remove products and update stock levels.
   - **Data Export**: Export order data (CSV/Excel) for offline processing.
4. **Secure Access**: Middleware-protected dashboard to ensure only authorized personnel can manage data.
